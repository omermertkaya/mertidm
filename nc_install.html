Kanıtlar 3. NC Paketi Kurulumu ve Kontrol

Eksik Paket Kurulumu

Eksikliğin giderilmesi adına aşağıdaki komutları kullanarak netcat-openbsd paketinin kurulumunu gerçekleştirdim: 

sudo apt update
sudo apt install -y netcat-openbsd
Kontrolün Tekrar Çalıştırılması

Netcad ile ilgili yaptığım araştırmalarda sorunun paket adından kaynaabileceği ile ilgili yönlendirmelere rastladım. Yukarıdaki ekran görüntüsünde görüldüğü üzere sorunu çözmek adına birçok farklı yöntem denedim. Fakat paket ile ilgili olumlu bir sonuç elde edemedim. Bu durumda support ekibine başvurarak sorunu çözmeye çalışırdım. PoC kapsamında belirtilen “NT” ifadesinin netcat’i mi (nc / ncat) temsil ettiği konusunda emin olamadım. Farklı Linux dağıtımlarında netcat paketinin:

  • paket adı (netcat, netcat-openbsd, ncat)
  • binary adı (nc, ncat)
  • binary yolu (/bin/nc, /usr/bin/nc)
değişiklik gösterebildiğini gözlemledim.
